# admin-dashboard
Project: Admin Dashboard

I gave myself a two week limit for this project which is equivalent to about 10-12 hours at my normal pace. I got to practice my grid skills and layering grids within grids. I believe overall this project was not bad but a bit tedious to set up. I did enjoy figuring out how to set up sections that were similar to one another to have them all updated at the same time. Overall, I felt that this project just reinforced how grids interact with one another and what things you should look for.
